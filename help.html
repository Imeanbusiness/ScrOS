Scr-OS

Welcome to the Scr-OS documentation. All commands will be documented here.

Commands:

Customization:

>bgc (filename): Open a file to set as the background. NOTE: Image must be named exactly (without any folders, just the name) and put in the Images folder.

>fontcolor (hex code or color name): Set a specific font color to your liking

>outlcolor (hex code or color name): Set a specific font outline to your liking

>iconpack (icon pack folder name): Set a custom icon pack for apps on the home screen! Make sure that each icon image is SPECIFICALLY Matched to the original.

>font (font): The list of available fonts will be listed here: Montserrat, Nunito, Sans, Roboto, Ubuntu, Poppins, Homenaje, Pixelfy Sans, Raleway, Bebas Neue. More to come in the future!

Assistance:

>help: Get help

>kill: Quits Scr-OS

>ms (query): (Music) Search for a song on Spotify

>yt (query): Search for a Youtube video.

>gh: Open Scr-OS github

>gs (query): Google Search the query.

>ops (site): Open the site quickly

Productivity:

>calc (statement): Runs off of IMB Composite Software. Calculates statements

>compchk (integer): Runs off of IMB Composite Software. Finds factors.

System:

>fclock: Change clock format from 24 hour to 12 or vice versa.

>status: Check ScrOS status

>reset: Resets ScrOS memory. Run twice to confirm.
